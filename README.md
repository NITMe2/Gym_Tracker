# IronLog

A minimalist gym tracking app for Android. Log your lifts, track progress, and stay consistent.

## Download

**[Download the latest APK → GitHub Releases](https://github.com/NITMe2/Gym_Tracker/releases/latest)**

IronLog is distributed as a sideloadable Android APK. To install:
1. Download `IronLog.apk` from the releases page
2. Enable **Install from unknown sources** in your Android settings
3. Open the downloaded file to install

## Tech Stack

- React + Vite (web app bundled for Android via Capacitor)
- Local-first storage — no account required
